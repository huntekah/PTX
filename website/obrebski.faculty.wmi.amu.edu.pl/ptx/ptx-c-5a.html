Przetwarzanie tekstu
ćwiczenia 5

temat: Thrax

Zadania

  1. Przejrzeć przykłady z wykładu.
  2. Zbudować przetwornik poprawiający błędy ortograficzne.
  3. Stworzyć przetwornik, który będzie poprawiał błędy polegające na nieprawidłowym użyciu form przyimka z/ze, w/we, np: 
    przyjdę we wtorek -> przyjdę we wtorek
przyjdę w wtorek -> przyjdę we wtorek
jestem we kurtce -> jestem w kurtce
mam problem ze zadaniem -> mam problem z zadaniem
mam problem ze pamięcią -> mam problem z pamięcią

Wskazówki

  1. kompilacja pliku .grm 
        $ thraxmakedep NAZWAPLIKU.grm
    $ make
  2. test przetwornika 
    
    $ thraxrewrite-tester --noutput=10 --far=corrector.far --rules=corrector
    $ thraxrewrite-tester --noutput=10 --input_mode=dict.tsv --output_mode=dict.tsv --far=parser.far --rules=parse
